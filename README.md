# GrimDM — Voice AI Dungeon Master

A real-time voice-interactive AI Dungeon Master powered by Google's Gemini Live API. Dark fairy tale RPG where players speak naturally, NPCs have distinct character voices, scenes render as dark fantasy art, and ambient music sets the mood.

Built for the [Gemini Live Agent Challenge](https://devpost.com/) on Devpost.

## Features

- **Voice DM**: Speak naturally with the Dungeon Master using Gemini's native audio (sub-second latency, barge-in support)
- **NPC Character Voices**: Each NPC has a unique voice generated via voice-design API
- **Scene Illustrations**: Dark fantasy art generated by Imagen 4 on location changes
- **Background Music**: Mood-reactive ambient music loops that crossfade with gameplay
- **3-Channel Audio Mixing**: DM voice + NPC voice + background music, mixed in-browser
- **Persistent Game State**: Player stats, inventory, quests, and NPC relationships
- **Full RPG Mechanics**: Dice rolls, combat, inventory, quests, NPC relationships

## Architecture

```
Browser (Web Audio API, 3 audio channels)
  |
  +-- WebSocket (16kHz PCM bidi) --> Cloud Run (FastAPI + ADK)
  |                                      |
  |<- HTTP (images, NPC audio) ----------+---> Gemini 2.5 Flash Native Audio (DM voice)
  |                                      +---> Imagen 4 (scene art)
  |                                      +---> ScrappyLabs API (NPC voices)
  |                                      +---> Firestore (game state)
```

## Tech Stack

| Component | Technology |
|-----------|-----------|
| Voice Engine | Gemini 2.5 Flash Native Audio via ADK |
| Agent Framework | google-adk (Python) |
| NPC Voices | ScrappyLabs voice-design API |
| Scene Art | Imagen 4 via google-genai SDK |
| Backend | FastAPI + Cloud Run |
| Frontend | Vanilla HTML/CSS/JS + Web Audio API |

## Quick Start

```bash
cd backend

# Create .env from template
cp .env.example .env
# Edit .env with your GOOGLE_API_KEY

# Install dependencies
pip install -e .

# Run locally
uvicorn app.main:app --reload --port 8080
```

Open http://localhost:8080 — click "Speak" to start voice mode, or type in the text box.

## Deploy to Cloud Run

```bash
# Authenticate with GCP
gcloud auth login
gcloud config set project scrappylabs

# Deploy
bash deploy/deploy.sh
```
